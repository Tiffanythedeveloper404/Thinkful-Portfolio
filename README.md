# Thinkful-Portfolio
This is a personal portfolio website for Pete Thinkful, an artist based in Denver, Colorado. The site showcases Pete's artistic interests, featured works, and contact information.

## About the Project
The website is built using HTML and CSS and includes three main sections:

- **About**: Introduction to Pete and his artistic passions.
- **Portfolio**: A gallery of selected artworks with descriptions.
- **Contact**: Social media links and a message inviting collaboration.

## Features

- Clean, responsive layout
- Centered headings and images for visual clarity
- Semantic HTML structure for accessibility
- Styled lists and links for easy navigation
